# PromptMarket

**优质Prompt模板市场 - 发现、分享、出售AI Prompt模板**

## 功能

- ✅ Prompt模板库
- ✅ 分类浏览
- ✅ 搜索功能
- ✅ 免费/付费模板
- ✅ 一键复制
- ✅ JSON下载

## 快速开始

```bash
pip install -r requirements.txt
python src/main.py
```

访问：http://localhost:8001

## API

- GET /api/prompts - 获取Prompt列表
- GET /api/prompt/{id} - 获取单个Prompt
